Project Name: Smart Attendance Logger

Description: This is a mobile application developed to improve classroom attendance management. The app allows teachers to generate a QR code for each class session. Students scan the QR code using their phone camera to mark their attendance.

After scanning, the application verifies the student’s location using GPS to ensure they are physically present in the classroom. Attendance is recorded only if the student is within the allowed classroom area.

The system helps reduce manual attendance errors and prevents fake or remote check-ins.
